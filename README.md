# 🎮 Retro Video Games Analytics Dashboard

An interactive Power BI dashboard analyzing retro video game sales, genres, platforms, and regional performance. The project provides business insights into console popularity, game distribution, and sales trends using interactive visualizations.

## 🚀 Features

- Executive dashboard with key KPIs
- Interactive filters (Year, Genre, Publisher)
- Drillthrough page for console-level analysis
- Custom report tooltip
- Dynamic console images
- Sales by region analysis
- Genre-wise game distribution
- Release year sales trend
- Top-selling games by console

## 📈 Key Insights

- PlayStation 2 generated the highest global sales among the top consoles.
- North America contributed the largest share of sales for most platforms.
- Action and Sports games dominate the retro gaming market.
- Sales peaked during the early-to-mid 2000s before gradually declining.
